Welcome to UML Designer

UML Designer provides a set of common diagrams to work with UML 2.4 models. The intent is to provide an easy way to make the transition from UML to domain specific modeling. This way users can continue to manipulate legacy UML models and start working with DSL. Users can even re-use the provided representations and work in a total transparence on both UML and DSL models at the same time.

These plugins are released under the EPL Open-Source License.
